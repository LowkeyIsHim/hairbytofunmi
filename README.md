# HairByTofunmi - Premium Hair Stylist Website

This is a production-ready, full-stack website built with **Next.js 14 (App Router)**, **Tailwind CSS**, and **Supabase** for a real-time, premium online presence.

## Features

* **Premium & Feminine UI:** Elegant design with subtle animations.
* **Real-time Data:** Hair styles and site settings update instantly via Supabase Realtime.
* **Secure Admin Panel:** Login via Supabase Auth (Magic Link or Password).
* **Zero-Cost Deployment:** Frontend deployable on Vercel, backend uses Supabase Free Tier.
* **WhatsApp Booking:** All services link directly to a pre-filled WhatsApp message.
* **No File Storage Required:** Styles use image URLs provided by the admin.
